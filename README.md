Esse projeto foi criado 100% com intuito de aprender e evouluir os conhecimentos na linguagem de programação Python.

Nós fomos desde conceitos básicos com lógica simples.
Pasasamos por orientação a objetos, herança e polimorfismo
E terminamos com a criação de ambientes virtuais, uso de API e criação de arquivos.

O projeto se baseia em um aplicativo de restaurantes, no qual se pode cadastrar restaurantes, pratos com seus valores e descrição, e também podendo consultar os mesmos. T
udo foi feito com a linguagem de programação Python e FastApi.
